# Предиктивный анализ различных наборов

Кликабельный HTML/JS-прототип для GitHub Pages.

## Что внутри
- выбор критериев по наборам:
  - Объекты ОДХ
  - Работы на ОДХ
  - Работы на ОКБ
- поддержка операторов:
  - И / ИЛИ / НЕ
  - равно / не равно
  - больше / меньше
  - от / до
  - пусто / не пусто
  - содержит / не содержит
- базовые и пользовательские весовые коэффициенты
- имитация запуска анализа с прогрессом
- аналитические плашки
- карта на Leaflet + OpenStreetMap
- реестр с выгрузкой в Excel

## Файлы
- `index.html` — разметка
- `styles.css` — стили
- `data.js` — демо-данные и метаданные атрибутов
- `app.js` — логика приложения

## Как запустить локально
Открой `index.html` в браузере.

## Как выложить на GitHub Pages
1. Создай репозиторий
2. Загрузите эти 4 файла в корень
3. В Settings → Pages включи публикацию из ветки `main`, папка `/root`
4. Открой выданную ссылку
