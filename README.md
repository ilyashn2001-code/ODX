# Центр геоаналитических инструментов

Обновлённый HTML/JS-прототип для GitHub Pages.

## Что внутри
- шаг 1: выбор инструмента без лишнего описания сценария
- шаг 2: выбор анализируемого и сравниваемых наборов через раскрывающиеся списки с чекбоксами
- шаг 3: критерии по реальным атрибутам CSV
- шаг 4: параметры результата:
  - нормативный срок службы
  - процент непокрытой площади
  - примерный объём работ
  - примерная стоимость
- шаг 5: проверка конфигурации перед запуском
- после расчёта:
  - отдельный верхний контейнер приоритизации
  - карта по WKT-геометрии
  - итоговая таблица со скроллом только внутри таблицы

## Важно
Во всех CSV используется разделитель `;`.

## Файлы
- `index.html` — разметка
- `styles.css` — стили
- `app.js` — логика приложения
- `data_fixed.js` — данные, собранные из CSV
