# Центр геоаналитических инструментов

Обновлённый HTML/JS-прототип для GitHub Pages.

## Что изменено
- критерии строятся по реальным атрибутам из CSV-файлов
- поддерживаются три набора:
  - Набор ОГХ
  - Набор благоустройства (СОК)
  - Набор благоустройства (МР)
- в таблицу результата выводятся все строки и все атрибуты из файла `Результат анализа.csv`
- на карте отображается WKT-геометрия из столбца `Геометрия неблагоустроенной территории`
- приоритизация влияет на порядок строк результата и цвет объектов на карте

## Важно
Во всех CSV используется разделитель `;`

## Файлы
- `index.html` — разметка
- `styles.css` — стили
- `app.js` — логика приложения
- `data_fixed.js` — данные, собранные из CSV
