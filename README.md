# Центр геоаналитических инструментов

Сборка с доработками:
- возвращена прежняя логика шага «Критерии» с группировкой/разгруппировкой строк
- в сравнительных наборах доступен и «Набор ОГХ»
- шаг проверки остаётся перед результатом, а новый анализ запускается отдельной кнопкой
- системная приоритизация задана заранее и редактируется только по переключателю
- карта строится по WKT из результата анализа
- таблица результата прокручивается горизонтально внутри своего блока
