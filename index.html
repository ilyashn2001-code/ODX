Выберите основной анализируемый набор и один или несколько наборов для сравнения.

Шаг 3. Критерии отбора данных

Для каждого выбранного набора настраиваются собственные условия. Внутри группы действует связь И или ИЛИ, а сами группы внутри блока объединяются через ИЛИ.

Шаг 4. Приоритизация

Необязательный этап. Если не включать ручную настройку, используются базовые коэффициенты.
