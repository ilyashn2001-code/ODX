Шаг 1. Наборы данных

Сначала выбери, с какими наборами работает пользователь. Только выбранные наборы будут доступны в критериях.

Для запуска анализа должен быть выбран хотя бы один набор.

Шаг 2. Весовые коэффициенты

Коэффициенты влияют только на приоритет объектов, а не на состав выборки.
