Шаг 1. Выбор инструмента

Сначала выберите сценарий. Для рабочего кейса используйте инструмент предиктивного анализа.

Шаг 2. Выбор наборов данных

Выберите основной анализируемый набор и один или несколько наборов для сравнения.

Основной

Анализируемый набор

Итоговый реестр строится именно по этому набору.

Сравнение

Наборы для сравнения

Отображаются все наборы, кроме выбранного в анализируемом блоке.

Шаг 3. Критерии отбора данных

Для каждого выбранного набора создаётся корневая группа. Выделяйте строки и подгруппы галочками, объединяйте их через «Сгруппировать», задавайте связку И / ИЛИ и при необходимости включайте НЕ для строки или группы.

Шаг 4. Приоритизация

После получения результата блок остаётся доступным: можно менять коэффициенты и сразу перестраивать список результата.

Шаг 5. Запуск расчёта

После настройки наборов, критериев и приоритизации запустите обработку. В финале останутся карта, реестр и приоритизация.
