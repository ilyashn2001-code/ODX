Шаг 1. Выбор инструмента

Сначала выберите сценарий. Для рабочего кейса используйте инструмент предиктивного анализа.

Шаг 2. Выбор наборов данных

Выберите основной анализируемый набор и один или несколько наборов для сравнения.

Шаг 3. Критерии отбора данных

Для каждого выбранного набора доступны реальные атрибуты из CSV. Строки и группы можно выделять и объединять во вложенные блоки.

Шаг 4. Приоритизация

После получения результата блок остаётся активным: можно менять пользовательские коэффициенты и перестраивать порядок объектов.

Шаг 5. Запуск расчёта

Фильтрация подготавливает пользовательский сценарий, а таблица результата и карта строятся по файлу результата анализа.
